bool isFibonacciSequence ( const vector< int > & sequence,
int & errorIndex )

Проверяет, является ли последовательность числами Фибоначчи.

Последовательность должна быть такой, что каждый элемент начиная с третьего равен сумме двух предыдущих.

Parameters
sequenceПоследовательность целых чисел (уже проверена на валидность).
errorIndexИндекс первого элемента, нарушающего последовательность Фибоначчи.
Returns
true, если последовательность корректна по Фибоначчи; false — иначе.

◆ validateInput()

int validateInput ( ifstream & inputFile,
ofstream & outputFile,
vector< int > & sequence )

Проверяет и считывает данные из входного потока.

Валидирует формат строки, количество строк, допустимость символов, количество чисел и диапазон значений. При успешной валидации заполняет вектор чисел из строки.

Parameters
inputFileВходной поток (например, std::ifstream input("input.txt")).
outputFileВыходной поток для сообщений об ошибках (например, std::ofstream).
sequenceВектор, в который будут сохранены целые неотрицательные числа.
Returns
int Код завершения:
  • 0 — успешно
  • 3 — пустой файл
  • 4 — более одной строки
  • 5 — недопустимые символы или отрицательные числа
  • 6 — менее двух чисел
  • 7 — более 1000 чисел
